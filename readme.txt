== Description ==

Concretá más ventas con entregas superrápidas, agilizá tu operación y ofrecé una experiencia de compra sin fricciones.

* Envíos, cambios y devoluciones en el día
* Logística inversa simplificada y en solo un click
* Seguimiento en tiempo real, de punta a punta
* Integrado con tu tienda WooCommerce

Empezá hoy mismo haciendo click [acá](https://www.jipink.com/#contacto).

Conocé más sobre nuestra integración para WooCommerce en este [tutorial](https://academy.jipink.com/docs/integraciones/integracion-con-woocommerce).

¿Tenés dudas? ¡[contactanos](https://www.jipink.com/#contacto)!

== Installation ==

1. Instalar el plugin desde el directorio de plugins de WordPress.
2. Activar el plugin desde la sección de plugins instalados de WordPress.
3. Configurar las zonas de envío en WooCommerce > Ajustes > Envío.
4. Configurar el plugin en la sección Jipink > Ajustes.
